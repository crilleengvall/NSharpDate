NsharpDate

NSharpDate is a category for the NSDate class that adds a few properties and methods.

How To Get Started

Download NSharpDate Add NSDate+NSharpDate.h and NSDate+NSharpDate.m from the NSharpDate folder to your project.

Installation with CocoaPods

CocoaPods is a dependency manager for Objective-C, which automates and simplifies the process of using 3rd-party libraries. pod 'NSharpDate', '1.0.1'
